EmperorOS Installation Guide

Follow these simple steps to install EmperorOS and turn your Fedora system into a powerful, modular meta-distribution.

1. Clone the Repository

First, you'll need to clone the official EmperorOS repository from GitHub:

git clone https://github.com/EmperorOS/empinstall.git

2. Navigate to the Directory

Move into the cloned repository:

cd empinstall

3. Run the Installation Script

Once inside the repository, you can run the installation script:

./empinstall.sh

This script will transform your existing Fedora system into an EmperorOS-powered system.

Once the installation is complete, you're ready to start using EmperorOS!
